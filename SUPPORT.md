# Support

OSS Maintainer Evidence is maintained as an open-source project without a guaranteed response time.

Use [GitHub Issues](https://github.com/kilmidas/oss-maintainer-evidence/issues) for reproducible bugs and scoped feature requests. Include the released version, Node.js version, GitHub CLI version, command shape with public placeholder names, exit code, and sanitized error message.

The support scope is the latest release collecting public GitHub.com data. Help with private or enterprise repository access, grant strategy, application decisions, custom data analysis, credential recovery, and unpublished forks is outside project support. The tool does not decide grant or program eligibility.

Never include tokens, private URLs, private report contents, email addresses, or application correspondence in an issue. Report exploitable vulnerabilities through [SECURITY.md](SECURITY.md).
